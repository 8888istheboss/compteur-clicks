# Counter Demo

Simple click counter demo ready to deploy on Railway.

## Deploy on Railway

1. Push this folder to a GitHub repository.
2. Open Railway and create a new project.
3. Choose "Deploy from GitHub repo" and select the repository.
4. Railway should detect it as a Node app automatically.
5. The app will start with the command defined in package.json.

## Local run

```bash
node server.js
```

Open http://localhost:3000
